# IoT Gateway to InfluxDB API

## What is this?

As of IoT gateway 3.6, the software does not support influxdbv2.
This API was created to bridge the gap from IoT gateway to influxDB. 

command to install influxdb on PR21:
sudo snap install influxdb --edge
## Folders

### API_Version
This contains the node-red flows to configure

**What is an API?**
API is the acronym for Application Programming Interface, which is a software intermediary that allows two applications to talk to each other.

### Dashboard_Versions

This contains the influxDB dashboards example
